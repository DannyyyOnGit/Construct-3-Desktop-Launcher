[h2]Software Description[/h2]
This software makes it easier to switch between Construct 3 Desktop Stable/Beta releases. Works with any release channel version of Construct 3 desktop on Windows.

[h2]Construct 3 Launcher - Instructions[/h2]
How to install and use the Construct 3 Desktop Launcher:
[ol]
    [li]Download and unzip all files and folders[/li]
    [li]Directly paste the C3 launcher executable and dependencies folder into your Construct 3 Desktop folder (where the Construct3.exe is)[/li]
    [li]Execute the launcher and pick your release (stable/beta)[/li]
    [li]Enjoy your Construct 3 Desktop experience with release control[/li]
[/ol]

[h2]Construct 3 Launcher - FAQ[/h2]
What are the system requirements?
[i]Windows 95+ ; ~3mb disk space; 25mb RAM; DirectX 9.0c[/i]

Do I always have to use the C3 launcher now?
[i]No! You just have to launch the stable/beta release once. Construct 3 Desktop will always run based on that decision.[/i]

Can I keep launching Construct 3 Desktop by running directly running project files?
[i]Yes! Once you've launched the release version your like, you can keep running Construct 3 Desktop by directly clicking on the executable or project files (e.g. c3p file).[/i]

Why do I need this, what has been made more simple?
[i]As things currently are, Construct 3 Desktop user are required to opt into either the Stable channel or Beta Channel version of C3 Desktop. This launcher makes things easier and doesn't require any specific release channel.[/i]


The Construct 3 Desktop Topic offers 2 release channels, which one do I need?
[i]It doesn't matter. The launcher works with any release channel version of Construct 3 Desktop.[/i]
