[h2]Description[/h2]
This software makes it easier to use the desktop build of Construct 3. Select custom release versions, enable developer features and automatically keep everything up to date.

[h2]Features:[/h2]
[ul]
[li]Install the official Construct 3 build or custom builds[/li]
[li]Run any stable, beta or selected release versions[/li]
[li]One-click update any installed Construct 3 builds[/li]
[li]Use advanced features like Chromium-Args or enable Safe Mode.[/li]
[/ul]
